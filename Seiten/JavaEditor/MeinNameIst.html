Mein Name ist..

×

ZULETZT ANGESEHEN

    Die Aufgabe der heutigen Stunde ist Mein Name ist.. Diese Aufgabe ist ein genaues replikat der Aufgabe HelloWorld. Hier müssen wir erneut einen Text ausgeben, nur mit etwas mehr Personalisierung. Die Aufgabenstellung lautet:

    Bevor es richtig losgeht müsst du deinen Namen und dein Computersystem auf der Konsole ausgeben können. Ich kann dir keine Jobs auf der Reise vermitteln, wenn du nicht in der Lage sind deinen Namen in Java auszugeben. Schreibe ein Programm mit dem Dateinamen "Vorstellung.java", das folgende Ausgabe erzeugt:
    CONSOLE: Meine Name ist Max Mustermann. Ich habe einen Laptop mit Windows 10.
    Ersetze Max Mustermann und das Computersystem mit Betriebssystemangabe durch deine eignen Angaben.

    Lösungsprozess der Aufgabe:

    Java 
    
                /**
                *
                * Das fertige Java Programm "Vorstellung.java"
                *
                * @version 1.0 vom 25.02.2024
                * @MeinInfoKurs.github.io
                */
    
                public class Vorstellung {
    
                public static void main(String[] args) {
                        System.out.println("Mein Name ist Max Mustermann");
                        System.out.println("Ich habe einen PC mit Windows 10.");
                    }
                }

    In der Aufgabenstellung wird gefordert, dass wir folgende Funktionen in unseren Code implementieren:

    1. Ausgabe unseres Namens
    2. Ausgabe unseres Betriebssystemes

    Konsolenausgabe

    In dieser Aufgabe ist nur die Ausgabe eines "Strings" (String = Text) erforderlich. Dies ist sehr einfach! Wenn du auf deiner Tastatur die Tastenkombination STRG + U drückst, dann wird ein System.out.println() automatisch erzeugt. Wenn du in die Klammern nun "Mein Name ist Max Mustermann schreibst, wird dieser String (String = Text) in der Konsole ausgegeben. 


    
                System.out.println("Mein Name ist Max Mustermann");
                System.out.println("Ich habe einen PC mit Windows 10.");
