Die 10 Bewährungsfragen

  1. Gib an, wie die Angabe des Datentyps und des Namens einer Variablen in Java genannt wird.

    2. Antwort:   

    Deklaration

  2. Beschreibe, was mit der Initialisierung einer Variablen gemeint ist und nenne ein Beispiel.

    3. Antwort:   

    Die Initialisierung bedeutet, einer deklarierten Variable einen Anfangswert zuzuweisen. Beispiel: int zahl = 10;

  3. Beschreibe wofür die Abkürzung EVA steht.

    4. Antwort:   

    Eingabe -> Verarbeitung -> Ausgabe

  4. Gib ein Beispiel für eine Zuweisung an eine Variable mit einer Division an.

    5. Antwort:   

    double ergebnis=10.0/2.0

  5. Beschreibe, wo im Code die Variablen deklariert werden.

    6. Antwort:   

    Variablen werden immer ganz oben (als erstes) im Code deklariert

  6. Gib an, wie wird das Zusammenfügen von Zeichenketten mit dem Plus-Operator genannt.

    7. Antwort:   

    Konkatenation

  7. Gib an wie der Datentyp für Zeichenketten genannt wird.

    8. Antwort:   

    String

  8. Gib an wie der Datentyp für Dezimalzahlen genannt wird. Gebe ein Beispiel einer Deklaration mit einer Initialisierung an.

    9. Antwort:   

    double zahl=2.49

  9. Gib den Shortcut für System.out.println() an.

    10. Antwort:   

    STRG + U

  10. Gib den Datentyp für die Wahrheitswerte true und false an.

    11. Antwort:   

    Boolean
